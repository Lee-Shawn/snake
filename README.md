# 贪吃蛇

## 功能

### 1.0 版本

* 蛇吃掉食物，增加得分
* 撞到墙壁，游戏结束
* 撞到自己身体，游戏结束
* 开始/暂停 功能
* 菜单栏重新开始功能

### 2.0 版本

* 优化游戏界面
* 添加背景图片
* 添加背景音乐
* 添加游戏设置功能
* 添加配置文件
* 添加游戏排名
